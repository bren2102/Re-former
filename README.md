# Sample App

This a ruby on rails based application that showcases the benefits of Rails framework to generate form elements with some useful helpers like form_tag, form_form, text_field, etc.

# Requirements

- Ruby > 2.5
- Ruby on Rails > 5.2
- Rubocop

## Built With

- Ruby
- Ruby on Rails
